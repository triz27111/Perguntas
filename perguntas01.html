Respostas

1. Qual é o seletor CSS usado para selecionar um elemento pelo seu ID?

Resposta: se usa o "#" acompanhado do elemento escolhido.

2. Como selecionar todos os elementos de uma determinada classe com CSS?

Resposta: colocando um "." antes de colocar o nome da classe.

3. Qual é o seletor CSS usado para selecionar o primeiro filho de um elemento?

Resposta: first-child, ele seleciona o primeiro elemento filho de seu pai.

4. Como selecionar o último elemento filho de um elemento especifico usando CSS?

Resposta: last-child, ele seleciona o último elemento entre os filhos de um elemento pai.

5. Como selecionar todos os elementos filhos diretos de um elemento usando CSS?

Resposta: você pode usar o seletor de filho direto (>).

6. Qual é o seletor CSS usado para selecionar todos os elementos irmãos imediatos após um determinado elemento?

Resposta: seletor adjacente, serve para selecionar o elemento imediatamente depois do primeiro elemento.

7. Como selecionar o primeiro elemento de uma determinada tag usando CSS?

Resposta: basta passar n + e o número do elemento a partir do qual você quer selecionar.

8. Qual é o seletor CSS usado para selecionar um elemento com base na sua posição em relação a outros elementos?

Resposta:nth-of-type(N), seleciona o elemento que seja o N-ésimo do seu tipo.
