1. Qual é o objetivo principal da propriedade flexbox no CSS?

Resposta: O Flexbox tem como meta ser um modo mais eficiente para criar leiautes, alinhar e distribuir espaços entre itens em um container, mesmo quando as dimensões destes itens são desconhecidas e/ou dinâmicas

3. Quais são as duas principais propriedades usadas para controlar o posicionamento dos itens flex dentro de um contêiner?

Resposta: flex-direction e flex-wrap.

4. Explique a diferença entre "flex-direction: row" e "flex-direction: column" em termos de posicionamento dos itens.

Resposta: Os itens são organizados em forma exibição em linha da direita para a esquerda; column: Os itens são organizados em forma de colunas iniciando de cima para baixo; column-reverse: Os itens são organizados em forma de colunas iniciando de baixo para cima.

5. Como você alinha os itens flex horizontalmente dentro de um contêiner?

Resposta: Você pode usar a propriedade align-items para alinhar itens ao longo do eixo transversal

6. Como você alinha os itens flex verticalmente dentro de um contêiner?

Resposta: Você pode usar a propriedade align-items para alinhar itens ao longo do eixo transversal. Se você usá-lo junto com a propriedade center, ele centralizará os itens verticalmente quando a direção for linha ou reversa, e horizontalmente quando a direção for coluna ou reverso da coluna.

8. Qual é a finalidade da propriedade "justify-content" em um contêiner flex?

Resposta: alinhar os elementos ao longo do eixo principal (main-axis), cuja direção (row ou column) é definida a partir da propriedade flex-direction.

9. Qual é a finalidade da propriedade "align-items" em um contêiner flex?

resposta: alinhar os elementos no eixo transversal.

10. Como você altera a ordem dos itens flex dentro de um contêiner?

Resposta: Para alterar a ordem dos itens flex dentro de um contêiner, você pode utilizar a propriedade 'order' do CSS. A propriedade 'order' permite especificar a ordem em que os itens flex são exibidos no contêiner, independentemente de sua ordem no código HTML.
