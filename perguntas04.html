1. Qual é o objetivo principal da propriedade position' no CSS?

Resposta: O objetivo principal da propriedade 'position' no CSS é controlar o posicionamento de um elemento em relação ao seu contexto de layout. Ela permite definir como um elemento deve ser posicionado na página, levando em consideração o fluxo normal dos elementos ou alterando-o de forma mais específica.

3. O que acontece quando você define 'position: static em um elemento?

Resposta: Quando você define position: static em um elemento, ele segue o fluxo normal do layout, sem sofrer nenhum posicionamento especial. Essa é a configuração padrão para a propriedade 'position'.

4. Como funciona o posicionamento relativo ('position: relative') em relação ao elemento pai?

Resposta :O posicionamento relativo (position: relative) em relação ao elemento pai permite posicionar um elemento deslocando-o em relação à sua posição normal no fluxo do layout, com base nas propriedades top, right, bottom e left.

5. Quais propriedades adicionais podem ser usadas em conjunto com 'position: relative para controlar o posicionamento de um elemento?

Resposta: z-index, margin, transform, overflow

8. O que acontece quando você define 'position: fixed' em um elemento?

Resposta: Quando você define position: fixed em um elemento, ele é posicionado de forma fixa em relação à janela de visualização do navegador. Isso significa que o elemento permanecerá no mesmo local, mesmo quando a página é rolada.

10. Como o valor position: sticky funciona e quando é apropriado utilizá-lo?

Resposta:A propriedade position: sticky é usada para criar elementos que "grudam" em uma determinada posição na janela de visualização quando o usuário rola a página. O comportamento do position: sticky combina características do position: relative e position: fixed.
